## 二、项目结构

```
app/src/main/java/
├── com/danmakuplayer/                         ← 应用代码
│   ├── MainActivity.kt                        # 单 Activity + Compose 导航 + 文件选择器
│   ├── PlayerViewModel.kt                     # ViewModel：播放器 + 弹幕生命周期管理
│   ├── manager/
│   │   ├── MpvPlayerManager.kt                # mpv 封装，实现 PlayerClockProvider + PlayerEventSource
│   │   └── MpvSurfaceView.kt                  # 自定义 SurfaceView，mpv 视频输出
│   ├── bridge/
│   │   ├── api/
│   │   │   ├── PlayerClockProvider.kt         # 播放器时钟接口
│   │   │   ├── PlayerEvent.kt                 # 事件 sealed class（6 种事件）
│   │   │   └── PlayerEventSource.kt           # 事件源接口 + PlayerEventListener
│   │   └── bytedance/
│   │       └── ByteDanceDanmakuAdapter.kt     # 适配器：mpv 事件 → ByteDance DanmakuController
│   ├── model/
│   │   ├── DanmakuItem.kt                     # 弹幕数据模型
│   │   ├── DanmakuType.kt                     # 弹幕类型枚举（ScrollRtl/TopFixed/BottomFixed/Special）
│   │   ├── DanmakuSourceType.kt               # 来源类型枚举
│   │   ├── DensityMode.kt                     # 密度模式（Strict/Balanced/Crowded）
│   │   └── GlobalDanmakuStyle.kt              # 全局样式（字号/速度/透明度/描边/密度）
│   ├── parser/
│   │   └── BilibiliXmlParser.kt               # Bilibili XML 弹幕解析器
│   └── ui/
│       ├── HomeScreen.kt                      # 首页：选择视频 + 加载弹幕文件
│       └── PlayerScreen.kt                    # 全屏横屏播放器 + 弹幕控制面板
│
└── com/bytedance/danmaku/render/engine/       ← 字节跳动弹幕引擎（第三方，保留原包名）
    ├── DanmakuView.kt                         # 弹幕渲染 View（继承 View）
    ├── control/                               # 控制层（DanmakuController/Config/Command/Event）
    ├── data/                                  # 数据层（DanmakuData/DataManager）
    ├── render/                                # 渲染层（RenderEngine/Layer/DrawItem/Cache）
    ├── touch/                                 # 触摸事件处理
    └── utils/                                 # 工具类（Logger/Constants/Profiler）
```

## 三、数据流

```
Bilibili XML 文件
    → BilibiliXmlParser.parse() → List<DanmakuItem>
    → ByteDanceDanmakuAdapter.loadData() → List<TextData>
    → ByteDance DanmakuController.setData()

mpv 播放事件 → MpvPlayerManager → PlayerEvent
    → ByteDanceDanmakuAdapter → DanmakuController.start/pause/clear
    → ByteDance DataManager（挂钟时间推进）
    → DanmakuView.draw()
```

## 四、可调节样式参数

| 参数 | 范围 | 默认值 | 说明 |
|------|------|--------|------|
| 字号 | 0.3x ~ 3.0x | 1.0x | 基准 48px，乘以缩放系数 |
| 速度 | 0.3x ~ 3.0x | 1.0x | 基准 8000ms 滚完全程，除以速度系数 |
| 透明度 | 0.1 ~ 1.0 | 0.85 | 映射到 ByteDance alpha 0~255 |
| 描边 | 0 ~ 8dp | 2dp | 映射到 ByteDance strokeWidth |
